Our brand represents the intersection of **Artificial Intelligence**, **Career Development**, and **Personal Growth**.

## ✨ Design Elements

### Glassmorphism Style
```css
background: rgba(255, 255, 255, 0.1);
backdrop-filter: blur(10px);
border: 1px solid rgba(255, 255, 255, 0.2);
```
**Usage**: Cards, modals, containers

### Gradients
**Linear Gradient (Primary)**
```css
background: linear-gradient(135deg, #6366f1 0%, #8b5cf6 50%, #a78bfa 100%);
```

**Linear Gradient (Accent)**
```css
background: linear-gradient(135deg, #ec4899 0%, #06b6d4 100%);
```

### Border Radius
- Small: 4px - Form inputs, small buttons
- Medium: 8px - Cards, medium buttons
- Large: 12px - Large containers
- Full: 9999px - Pill-shaped buttons

### Shadows
**Glow Effect**
```css
box-shadow: 0 0 20px rgba(99, 102, 241, 0.3);
```

**Subtle Shadow**
```css
box-shadow: 0 10px 30px rgba(0, 0, 0, 0.2);
```

## 🎯 Logo Usage Guidelines

### ✅ DO
- Use the logo with plenty of white space
- Maintain aspect ratio when resizing
- Use the gradient colors as specified
- Apply shadows/glows for depth

### ❌ DON'T
- Distort or stretch the logo
- Change colors without brand approval
- Use on backgrounds with similar colors (not enough contrast)
- Rotate or flip the logo

### Minimum Size
- Web: 64px
- Print: 1 inch (96px at 96dpi)
- Social Media: 200px (for profile pictures)
