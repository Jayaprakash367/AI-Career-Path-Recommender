💡 TIPS FOR CUSTOMIZATION

To add new career:
  1. Add row to data/careers.csv
  2. Run: python model\train_model.py
  3. Restart Flask server

To change theme:
  1. Edit tailwind.config.js colors
  2. Update index.css gradients
  3. Run: npm run dev (auto-reloads)

To add new form field:
  1. Update Form.jsx state
  2. Add UI input in appropriate step
  3. Update backend /predict endpoint

To deploy:
  1. Push to GitHub
  2. Connect to Render/Heroku
  3. Set environment variables
  4. Deploy!
